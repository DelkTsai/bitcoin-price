## 比特币价格，Chrome插件
> 正如你看到的，这个chrome插件主要是显示 比特币、黄金、美元对RMB的比例
> 如果你对区块链、前端技术，以及投资感兴趣，那就一起参与这个项目吧！！！

![./doc/bitcoin.png](./doc/bitcoin.png)

## 如何使用该插件
打开Chrome，更多工具（more tools），扩展（extensions），加载解压的扩展（load unpacked extension），选择bitcoin-price/src目录
